### 问题：

在linux系统下，使用ls命令时，会看到目录、普通文件、可执行文件都拥有不同的颜色，但是在使用Mac系统时，却没有。

在Ubuntu 15.04系统下执行命令：

```
type ls
```

会看到ls其实是

```
 ls --color=auto
```

的别名。

而在Mac系统下却没有这个参数。

### 解决办法：

查看Mac系统下man手册:

```
man ls
```

发现里面有关颜色的参数是 -G。

因此执行

```
ls -G
```

这样是有颜色区分了。

那么现在执行命令:

```
alias ls='ls -G'
```

这样，每次执行ls就都有颜色了，不过重启会失效。那么，现在写入到脚本中:

```
echo 'alias ls ="ls -G"' >> ~/.bash_profile
```

这样重启会就不用重新设置了
